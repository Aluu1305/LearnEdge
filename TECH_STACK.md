# LearnEdge Tech Stack

## Frontend
- **Framework:** React
- **Deployment:** Vercel

## Backend
- **Runtime:** Node.js
- **Framework:** Express.js
- **Deployment:** Railway or Render

## Database
- **Type:** PostgreSQL
- **Service:** Supabase (free tier)

## Payments
- **Processor:** Stripe
- **Features:** Checkout, Connect (for creator payouts), Tax

## Video Hosting
- **Storage:** AWS S3
- **Delivery:** CloudFront CDN

## AI Assistant
- **Provider:** Anthropic Claude API
- **Model:** Claude Sonnet

## File Storage (Images/Thumbnails)
- **Service:** Cloudinary or AWS S3

## Email
- **Service:** Resend or SendGrid

## Monitoring & Analytics
- **Errors:** Sentry
- **Analytics:** Google Analytics 4
- **User recordings:** Hotjar

## Other Tools
- **Version Control:** Git + GitHub
- **Code Editor:** VS Code
